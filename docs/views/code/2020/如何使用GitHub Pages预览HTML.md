本篇文章将教你如何使用Github Pages浏览你自己的HTML页面~

<!-- more -->

1. 注册Github账号
2. 新建一个仓库demo
3. 将仓库clone到本地，编写你自己的HTML页面
4. 使用git将完成的页面上传至仓库
5. 进入本仓库的Github Settings页面，选择在Source一项设置你的Github Pages所引用的分支
6. 设置完毕后即可访问Github自动生成的地址浏览你的页面了

> 注：HTML页面名必须为index.html
