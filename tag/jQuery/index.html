# jQuery如何获取元素

jQuery的第一步，往往就是将一个选择表达式，放进构造函数jQuery（简写为$），然后得到被选中的元素。 选择表达式可以是CSS选择器：

$(document) //选择整个文档对象

$('#myId') //选择ID为myId的网页元素

$('div.myClass') // 选择class为myClass的div元素

$('input[name=first]') // 选择name属性等于first的input元素
